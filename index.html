HOW TO PLAY

Goal: Certify three engineering bays on each MBT, then pass the live-fire exercise. Clear all three hulls.

1 — Stabilizer PID: Tune Kp, Ki, Kd so the gun tracks commanded azimuth under terrain torque. Run the certification slew (Space).

2 — Hydraulic traverse: Set pump pressure, relief crack, and flow gain so the traverse slews without pegging pressure or stalling.

3 — Fire-control bus: Fix CAN node IDs and 120 Ω termination. Run the diagnostic sweep until every node ACKs.

Live fire unlocks only when all three bays pass on that hull.

Keyboard: 1/2/3 switch bays · Q/E slew ±15° · Space run certification · Esc hangar

FIELD MANUAL

Stabilizer PID

u = Kp·e + Ki·∫e dt − Kd·ω — Kp for stiffness, Kd for damping, Ki for sustained bias rejection.

Hydraulic traverse

Pump builds line pressure; relief vents above crack pressure; flow gain maps position error to spool opening. Burst pressure fails certification.

Fire-control CAN bus

Nodes: Stabilizer 0x10, Traverse 0x20, Gunner 0x30, LRF 0x40. Both bus ends need 120 Ω termination or reflections corrupt frames.
